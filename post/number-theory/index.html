GCD problem

  • gcd(a,b) == gcd(a-b,b) (a>=b)
  • gcd(a1,a2,…,an) == gcd(a1,a1-a2,a1-a3,…,a1-an)
  • gcd(a,b) = gcd(b,a%b)
  • cpp -> __gcd

E sieve

外层循环从2到n 如果枚举到的i在前面的腥风血雨中均未被标记为非素数 则可以将他塞进素数组合 内层循环 在现有的素数p1 - pk 则 p1和当前的数的乘积必然不是素数 isprime标记为false。 一个边界要处理一下： i*pi 可能会越界要判断一下。

void initprime(int n)
{
    memset(isprime,1,sizeof isprime);
    isprime[0] = 0, isprime[1] = 0;
    for(int i=2;i<=n;++i){
        if(isprime[i]) prime.push_back(i); 
        for(auto x : prime){
            if(i*x>=n) break;
            isprime[i*x] = false;
            if(i%x==0) break; //x已经是i的最小质因数 如果x是i的最小质因数 则 x*i 的最小质因数也是x
        }
    }
}
